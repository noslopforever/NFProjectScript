# Abstract

# Usage
nfpsc --source "SourceRoot" [options]

options:
- --target
    - cpp
    - ts
    - c#
    - py
    - uecpp
    - unityc#
    - cocos
- --output-folder : default will be "%(SourceRoot)/output/%(language)"

# Dependencies
- [graphviz](https://graphviz.org/)
  - ubuntu : sudo apt install graphviz
